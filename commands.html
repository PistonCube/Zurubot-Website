📚 Command List

These are the available commands for ZuruBot. Use them to enhance your Discord server experience!

AI (2)

  • /chatgpt → 🤖 Ask a question to ChatGPT using GPT-4o
  • /chatgpt2 → 🤖 Ask a question to ChatGPT using GPT-4

CORE (8)

  • /commands → 📚 Shows the list of available commands.
  • /help → 📚 Help & Support
  • /ping → 🏓 Bot and api ping
  • /poll → 📊 Create a poll with up to 10 choices
  • /restart → 🔄 Restart the bot (Admin only)
  • /status → 📊 Give my status
  • /support → 🚀 Get support or join the official support server
  • /vote → 🗳️ Vote for ZuruBot

GENERAL (1)

  • /donate → 💸 Donate for ZuruBot

ECONOMY (8)

  • /balance → 💰 Check your coin balance
  • /daily → 🎁 Claim your daily coins (24h cooldown)
  • /deposit → 💰 Deposit coins into your bank
  • /admin-money → 💰 [ADMIN] Give money to a user
  • /leaderboard → 🏆 Top 10 richest users in the server
  • /rob → 🦹 Attempt to rob coins from another user
  • /transfer → 💸 Transfer coins to another user
  • /work → 💼 Work to earn coins (4h cooldown)

FUN (2)

  • /8ball → 🎱 Ask the magical 8-ball a question!
  • /randomfact → 📚 Get a random fact to expand your knowledge!

MINECRAFT (3)

  • /hypixelstats → 🟣 Complete Hypixel Statistics
  • /minecraftinfo → 📜 Complete Minecraft information
  • /minecraftserver → 🏓 Check the status of a Minecraft server (Java or Bedrock)

MODERATION (13)

  • /avatar → 🖼️ Display the avatar of a mentioned user 📸
  • /ban → 🔨 Ban a user with an optional reason
  • /embedmessage → ✨ Create a custom embed message with title, content, and optional settings 🎨
  • /invite → 🔗 Invite me to your server! 🚀
  • /kick → 🔨 Kick a user with an optional reason
  • /membercount → 📊 Get the total number of members in the server
  • /mute → 🔇 Mute a user with an optional reason
  • /purge → 🧹 Clear a specified number of messages from the channel
  • /purgeuser → 🧹 Clear messages from a specific user in the current channel
  • /serverinfo → 🌐 Get information about the server
  • /unban → 🔓 Unban a user with an optional reason
  • /unmute → 🔊 Unmute a user
  • /userinfo → 👤 Get information about a user

MUSIC (20)

  • /back → 🔙 Go back to the previous song
  • /clear → 🗑️ Clear all the music in the queue
  • /controller → 🕹️ Set controller channel
  • /filter → 🔁 Add a filter to your track
  • /jump → 🆙 Jumps to particular track in queue
  • /loop → 🔁 enable or disable looping of song's or the whole queue
  • /nowplaying → ▶️ view what is playing!
  • /pause → ⏸️ pause the track
  • /play → ▶️ play a song!
  • /playnext → ⏯️ song you want to playnext
  • /queue → ⏳ Get the songs in the queue
  • /remove → ⛔ remove a song from the queue
  • /resume → 📍 play the track
  • /save → 📌 save the current track!
  • /search → 🔎 search a track
  • /seek → 👑 skip back or foward in a song
  • /shuffle → 🔀 shuffle the track
  • /skip → ⏩ stop the track
  • /stop → ⛔ stop the track i put
  • /volume → 🔊 adjust

UTILITY (4)

  • /calculator → 🧮 Interactive calculator with buttons
  • /color → 🎨 Convert between color formats (HEX, RGB, HSL)
  • /remindme → ⏰ Set a reminder and get notified later
  • /timestamp → ⏳ Generate/convert timestamps for development
